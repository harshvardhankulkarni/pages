Vendors Form: Vendors

Create vendors first — every PO and expense references a vendor.
▸

Overview & Purpose

▸

Vendors stores your suppliers — companies that sell materials, equipment, and services. Create before POs or expenses.

Rule: Create each vendor before recording a PO or expense against them.

TC-1A-01

Create Vendor with Contacts & Documents

▸
Steps:
  1. Open Vendors form
  2. Enter: Vendor Name = TechSupply Co., Email = orders@techsupply.com, Payment Terms = Net 30
  3. In Vendor_Contacts subform: add Raj Patel (Primary) and Priya Sharma
  4. In Vendor_Documents: upload GST Certificate
  5. Submit
Expected: Record saved with auto-ID. Contacts and document visible.
Edge cases: Blank name → alert. 5+ contacts → all save. Large file → still uploads.
TC-1A-03

Create Account with Contacts & Documents

▸
  1. Open Accounts form
  2. Enter: Account Name = ABC Manufacturing, Payment Terms = Net 45
  3. Add contact: Vikram Singh (Primary)
  4. Upload Signed MSA document
  5. Submit
Expected: Record saved with contacts and document.
TC-1A-05

Create Project

▸
  1. Open Projects form
  2. Project Name = Office Renovation 2026, Account = ABC Manufacturing
  3. Start = 01-Jun-2026, End = 31-Dec-2026, Budget = 50000
  4. Status = Active, Submit
Expected: PROJ-0001 auto-generated.
Edge: End before Start → alert. Blank name → required.
TC-1A-07

Inventory Item — Goods with Stock

▸
  1. Open Inventory_Items form
  2. Item Name = Cement Bag (50kg), Type = Goods, Category = Raw Material
  3. Purchase Price = 8.50, Reorder Level = 100
  4. In Item_Warehouse_Stock: Main WH → 500, Secondary Store → 200
  5. Submit
Expected: SKU-0001. Stock Value = 500 × 8.50 = 4250. Available = 500.
