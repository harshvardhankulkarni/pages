**Creator notes:**
- System Type: Insert > Dropdown. Enter each option as `value|display`. The value (EP, PU, etc.) is used by Deluge as the code prefix. The display label is what users see.
- System Code: Insert > Text. Check "Required", "Unique" (Custom Error: "System Code already exists"), "Read Only". Do NOT use native Auto Number — Deluge generates the value.
- System Name: Insert > Text. Check "Required".
- Thickness: Insert > Text.
- Description: Insert > Text (Multi-line). Set Rows Visible = 4.
- UOM: Insert > Dropdown. Enter 2 options: `Sq.Ft`, `Sq.Mtr`. Check "Required".

## 3. Deluge Code for Auto-Number

### Trigger
**On Submit — Before record is added to the database**

### Workflow: Generate System Code

```deluge
// ─── Auto-Generate System Code ─────────────────────────────────
// Constructs {System Type prefix}-{3-digit sequential} code
// Example: EP-001, PU-002, DEM-001, ANTI-003

prefix = input.System_Type;

if(prefix != null && prefix != "")
{
    // Query existing records with same System Type, sorted descending by System Code
    last_records = zoho.creator.getRecords("chemsol", "System_Master",
        "System_Type == '" + prefix + "'",
        1, "System_Code desc");

    last_seq = 0;

    if(last_records.size() > 0 && last_records.get(0).get("System_Code") != null)
    {
        last_code = last_records.get(0).get("System_Code").toString();

        if(last_code.contains("-"))
        {
            last_seq = last_code.substringAfter("-").toLong();
        }
    }

    new_seq = last_seq + 1;
    formatted_seq = new_seq.toString().padLeft(3, "0");

    input.System_Code = prefix + "-" + formatted_seq;
}
else
{
    throw "Please select a System Type before saving.";
}
```

### Sequence generation examples

| System Type | Existing Codes | Next Code |
|-------------|---------------|-----------|
| EP | (none) | EP-001 |
| EP | EP-001 | EP-002 |
| EP | EP-001, EP-002 | EP-003 |
| EP | ... EP-009 | EP-010 |
| PU | (none) | PU-001 |
| PU | PU-001, PU-002 | PU-003 |
| DEM | (none) | DEM-001 |

Each System Type maintains its own independent sequence. All codes are zero-padded to 3 digits regardless of prefix length.

### Workflow setup in Creator
1. Go to Workflows tab within System Master form designer.
2. Click "New Workflow" > "On Submit" > "Before record is added to the Database".
3. Name: "Auto-Generate System Code".
4. Paste the Deluge code into the editor.
5. Save & Enable.

**Creator steps:**

5. Enable Search on: System Code, System Name.
6. Enable Summary > Record Count.
7. Set report permissions (inherits form Read access).
8. Save as "System Master List".
