Implementation Roadmap — 13 Phases (1A → 1L)

Phased rollout across all departments. Each phase corresponds to a vertical functional slice.

1A–1F Master Data Foundation Item Muster · Supplier · System · System Comp. · BOM · Customer · Store

Global shared data — Item Muster (6 categories), Supplier Master (16 fields), System Master (EP/PU/DEM codes), System Composition (System→FG mapping), BOM Formulation (FG→RM with ratios), Customer/Site Master, Store Master (6 warehouses).

2A–2E Procurement Suite PR · Rate Comp · PO · GRN · QC

Full procurement flow with Approval Matrix at PR stage, 5-supplier Rate Comparison, dual PO series (RM/RMWAD), partial GRN with checkbox, QC inspection with viscosity/density/color tests.

3A–3C Inventory Management MR · MIS · Return

Project-tagged Material Requisition, Material Issue Slip with balance tracking, Material Return processing. Stock auto-deducted on MIS posting.

4A–4D Production Planning · BMR · RM Consume · Packing

Production Planning with SO/WO autofetch, Batch Manufacturing Record, RM Consumption Entry, Packing Entry with FG stock update.

5A–5C FG & Logistics FGHM · Transport

FG Handover to client with QC data and inline acceptance, Vehicle & Transport tracking. Pop-up notifications on FGHM submission.

6A–6B Sales & Project Hub Sales Order · Project

Sales Order with system-code subtable (thickness, area, rate), auto-creates Project on Supply+Apply. Project is root entity — all downstream forms look up to it.

7A–7D Billing & Revenue Budget · Invoice · CCN · AR

Task Budget with 5 categories (Transport/Execution/Manpower/Tools/Overhead) + Actual columns for P&L. Customer Invoice, Credit Note, AR Receipt with GL auto-posting.

8A–8C Financial Close & GL Vendor Bill · AP · Journal

Vendor Bill/AP Payment, Supplier Credit Note (3 trigger scenarios), General Ledger with Journal Voucher. All transactions auto-post via 14C Rule. P&L auto-computed on Project Close.

9A–9D Governance & Reports Approval · Roles · Audit · Dashboards

Approval Matrix defining PR/PO limits per dept, User Roles & Permissions (Purchase/Sales/Store/Production/Finance/Admin/PM), Audit Log, 50+ reports and KPIs across all departments.
