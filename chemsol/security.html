User Access Form

Defined in User Access form: User Name (Zoho user lookup), Department (Purchase/Sales/Store & Logistics/Account & Finance/Admin/Project Coordinator/Project Manager 1/2/3), Role (Entry/Review/Approve/Admin), Email (autofetch), Status.
