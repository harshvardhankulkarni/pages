Project-Level Access Control

Users are assigned to projects via the Project Team configuration. A Project Manager can only see/edit forms tagged to their assigned projects. Key rules:

  • Project Manager 1/2/3 — Full CRUD on their assigned projects and tasks. View-only on procurement/inventory for their projects.
  • Project Coordinator — Create MR for their projects, view production and project status.
  • Sales — Create SO (which creates project). View all projects.
  • Purchase — See all projects' procurement data. No access to task management.
  • Store — See all projects' inventory movements. No access to project finances.

User Access Form

Defined in User Access form: User Name (Zoho user lookup), Department (Purchase/Sales/Store & Logistics/Account & Finance/Admin/Project Coordinator/Project Manager 1/2/3), Role (Entry/Review/Approve/Admin), Email (autofetch), Status.
