F Stock Management Rules
  • GRN posting — Qty added to stock only after posting; timestamp logged
  • MIS posting — Qty deducted from stock
  • FGAN approval — FG qty added to stock after acceptance
  • Material Return — Qty added back to stock
  • Min/Max stock — Alert when stock crosses thresholds
G P&L Calculation Rules Auto-computed on Project Close
  • On Project Close — P&L auto-calculated from all linked forms; snapshot stored
  • Real-time view — Live P&L accessible from Project dashboard at any time (not just at close)
  • Data completeness check — Before calculation, verify all GRNs are posted, all Task Budget Actuals filled, all BMRs closed
  • Margin threshold alert — If Gross Margin < 15%, notify Account & Finance + Project Manager
  • P&L snapshot — On project close, a frozen P&L record is created (no further changes)
  • Export — P&L report exportable as PDF (with company logo, project details, signatures)
