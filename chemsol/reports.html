Automation & Business Rules

Project-Centric Sequential Procurement Flow

SO (Supply+Apply)
  → Project Created (auto)
    → PR (tagged to Project)
      → Approval (as per Approval Matrix)
        → Rate Comparison (Purchase dept)
          → PO (Purchase dept — tagged to Project via PR)
            → GRN (Store dept — tagged to Project via PO)
              → QC/QA (QC dept)
                → Stock Updated (after GRN posting)

Material Issue Flow

MR (tagged to Project, created by Production/R&D)
  → MIS (Store dept issues material)
    → Stock Updated (deduction)

FG Flow

Production (tagged to Project) → FGHM (Production hands over FG)
  → Notification to Store/Logistics
    → FGAN (Store accepts FG)
      → FG Stock Updated (after approval)

Stock Management Rules

  • GRN posting — Qty added to stock only after posting; timestamp logged
  • MIS posting — Qty deducted from stock
  • FGAN posting — FG qty added to stock after acceptance approval
  • Material Return — Qty added back to stock
  • Min/Max stock — Alert when stock crosses thresholds (from Item Muster)
