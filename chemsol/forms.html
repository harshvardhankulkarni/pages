Forms & Module Structure

All transactional forms carry Project ID lookup. PR, PO, MR, GRN support N line items with autofetch — just like Zoho Books/CRM line items.

1 · Master Data Global — Shared Across Projects

Foundation for all operations. Like Zoho Books Items List and Zoho CRM Contacts — these are global lookups used by every project.

Central item repository for all RM, PM, FG, consumables. 6 categories: RM, Packaging Material, Tools & Consumable, Maintenance & Utility, Capital & Assets, Administration.

1B System Master Product Catalog

Flooring system definitions (EP01=1mm Epoxy, PU02=2mm PU). Code auto-format from prefix + number.

1C BOM Master Like Zoho Books BOM

Header + N line items: System Code → Item Code, UOM, Qty, Waste %. Auto Quantity Calculation formula.

1D Supplier Master Like Zoho Books Vendor

16 fields: Code, Name, Type, GSTIN, PAN, Contact, Bank, Payment Terms, Credit Days.

1E Customer/Site Master Like Zoho CRM Accounts

16 fields: Org, Contact, GST, PAN, Billing/Shipping, Site details.

1F Store Master + Bin Location Like Zoho Inventory Warehouses

Stores: Code, Name, Type (RM/FG/QC/Site), Location. Bin: Store → Rack → Shelf → Bin.

1G User Access + Approval Matrix Security Config

User → Department → Role (Entry/Review/Approve/Admin). Approval Matrix: Doc Type × Amount × Approver 1/2/3.

2D Goods Receipt Note (GRN) Partial via ✓ · N Items

PO ref → autofetch items. Checkbox per line for partial GRN. Stock updates only on posting.

GRN ref → inspect viscosity, density, color, moisture. Pass/Fail/Hold per item.

3 · Inventory / Stores Tagged to Project

Like Zoho Inventory: warehouse-based tracking with stock in/out/transfer. All tagged to project.

3A Material Requisition (MR) N Line Items

Project ID + items with autofetch available stock. Priority: Low/Medium/High/Urgent.

3B Material Issue Slip (MIS) N Items · Stock −

MR ref → autofetch items. Issued Qty, Balance Qty (formula). Stock deducted on posting.

3C FG Handover (FGHM) N Items · Notify Store

Project ID + client/site. FG products with QC status. Pop-up notification to Store on submit.

3D FG Acceptance (FGAN) Stock + After Approval

FGH ref → Damaged Qty, Accepted Qty (formula). FG stock updated after approval.

3E Additional Store Forms Return · Handover · Transport

Material Return (to stock), Material Handover (inter-store), Vehicle & Transport (delivery costs).

4 · Production Tagged to Project

4A Production Planning Per Project

Project ID, Period (Week/Month), SO/WO autofetch, FG Stock, Net Requirement = SO − FG.

4B Production Order + BMR N Items

MR ref, System Code, Batch No. Subforms: Raw Materials, Process Parameters, QC Results.

4C RM Consumption + Packing + Rework Per Project

Actual vs Standard comparison. FG packing with material consumption tracking. Rework with QC revalidation.

6 · Sales Creates Project

6A Sales/Work Order Master (SO) Dual Mode · N Line Items · Like Zoho CRM

Dual mode: Supply+Apply (auto-creates Project) / Supply Only (material sales).

Mode A: Supply+Apply

Customer details + N system line items (System Code, Thickness, Area, UOM, Rate, Amount). Footer: Total Amount, Payment Terms, Transport, Warranty, Commission.

Mode B: Supply Only

Customer details + N FG line items (FG Name, Qty, UOM, Rate, Amount). Footer: Total, Payment Terms, Transport.

Supply+Apply → auto-creates Project Like Zoho CRM Deal creation triggers downstream processes.
