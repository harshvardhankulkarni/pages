Project Hub + Sales Module

Root Entity Project is the hub — all downstream forms reference Project ID.

Sales/Work Order Master

Dual mode via selectbox: Supply+Apply (Work Order) / Supply Only (Material Sales). Supply+Apply → Auto-creates Project

Mode A: Supply+Apply

Fields: Date, Employee Name, Customer Code, Client Org Name *, Contact Person *, Contact No *, Alt Contact No, Email *, GST No *, PAN *, Regd Address *, Sales/Work Order No (autogen), Date, Site Name *, Site Address *, Site Manager, Contact No (Site), Project Type (Industrial/Commercial).

System subtable: System Code (lookup, autofetch name), Thickness, Area, UOM (conditional dropdown), Rate, Amount (Area×Rate).

Footer: Total Work Order Amount (formula), Payment Terms, Transportation Scope + Amount, Lead Time (Days), PO/BOQ Attachment (multiple), Warranty, Is Proper System Required (checkbox), Remark, Commission (checkbox → Percentage/Fix Amount + Amount).

Mode B: Supply Only (Material Sales)

Same customer fields + Sales Order No (autogen), Billing Address, Shipping Address.

FG subtable: FG Name (lookup), Quantity, UOM (autofetch), Rate, Amount (formula).

Footer: Total Sales Order Amount, Payment Terms, Transportation Scope + Amount, Lead Time, Remark.

Production Forms Tagged to Project

All production forms carry a Project ID lookup — production is planned and executed per project.

Production Planning

Planning No, Project ID, Period (Week/Month), Plant, SO/WO Qty (autofetch), FG Stock Available, Net Production Requirement = SO/WO Qty − Available FG. Status: Draft/Approved/Released.

Production Order

MR/Planning ref, Project ID, System Code, Planned Qty, Start/End Date. Status: Planned/In Progress/Completed/On Hold.

BMR Master

BMR No, Production Order ref, Project ID, Batch No, System Code. Subforms: Raw Materials, Process Parameters, QC Test Results. Yield/Output tracking.

RM Consumption Entry

BMR reference. Item-wise consumption. Actual vs Standard comparison. Per project.

Packing Entry

FG packing. Packing material consumption. Packed qty, batch numbers. Per project.

Rework Register

Rework reason, BMR ref, qty, material consumed, QC revalidation. Per project.
