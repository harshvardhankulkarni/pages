Procurement Flow

PR (Production/R&D)
  → Approval (Approval Matrix)
    → Rate Comparison (Purchase)
      → PO (Purchase — RM/RMWAD series)
        → GRN (Store — partial via checkbox)
          → QC/QA (QC dept — test results)
            → Stock Updated (after posting)

Material Issue Flow

MR (Production/R&D)
  → MIS (Store issues material)
    → Stock − (deduction on posting)

FG Handover Flow

Production → FGHM (FG Handover)
  → Pop-up Notification → Store/Logistics
    → FGAN (FG Acceptance)
      → FG Stock + (after approval)

Sales & Project Flow

SO/Work Order (Dual Mode)
  ├── Supply+Apply → Project → 5 Task Buckets
  └── Supply Only  → Material Sales (FG dispatch)

Project Execution Base:
  ├── Area Basis  → Application/Execution tasks shown
  └── Day Basis   → Manpower tasks shown
