Testing Guide Project Budget Tracking

Phase-by-phase test scenarios with sample data and expected results. Build, then test, then move to the next phase.

Workflow: Build Phase 1A → Run Phase 1A tests → Fix failures → Build Phase 1B → Run Phase 1B tests → Continue through Phase 1F → Run Integration Test.
▷

How to Use This Guide

▸
  1. Build the phase in Zoho Creator console
  2. Click the phase name in the sidebar
  3. Run each test scenario in order
  4. Record Pass/Fail using the Test Recording template
  5. Fix any failures before building the next phase
  6. After Phase 1F, run the Integration Test
▷

Structure of Each Test

▸

Each test scenario includes:

Steps
Numbered actions to perform
Sample Data
Values to enter in each field
Expected Results
What should happen after each step
Edge Cases
Failure conditions to verify
