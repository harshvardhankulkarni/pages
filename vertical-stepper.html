Automate business processes with Deluge workflows. These trigger on form submission, record update, or schedule.

Workflow Triggers

TriggerEventAction

// On-submit workflow
try {
  approval = zoho.creator.submitForApproval({
    recordId: input.ID,
    approver: "finance-head@company.com",
    message: "New budget requires approval"
  });
  sendEmail(approval.approver, "Budget Approval", emailBody);
} catch (e) {
  logError("Workflow failed: " + e.message);
}
6
Blueprints

Blueprints enforce approval flows with state transitions. Each transition can trigger validation and notifications.

Blueprint Stages

StageTransitions ToRequired Role
DraftPending ReviewCreator
Pending ReviewApproved / RejectedFinance Manager
ApprovedIn ProgressAuto
RejectedDraft (revision)Auto
Completed—Terminal
 
// Blueprint transition config
blueprint = zoho.creator.addBlueprint({
  module: "Budgets",
  stages: [
    { name: "Draft", next: "Pending Review" },
    { name: "Pending Review", next: ["Approved", "Rejected"] },
    { name: "Approved", next: "Completed" }
  ]
});
